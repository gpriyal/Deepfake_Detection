# Deepfake Detection using CNN
A binary image classifier to detect real vs AI-generated fake faces, built with TensorFlow/Keras.

## Dataset
[140k Real and Fake Faces](https://www.kaggle.com/datasets/xhlulu/140k-real-and-fake-faces) — 100k training images, balanced classes.

## Model
3-block CNN (Conv2D → BatchNorm → ReLU → MaxPool), followed by Dense layers with Dropout regularization.
Input: 128×128 RGB | Loss: Binary Cross-Entropy | Optimizer: Adam
Val Accuracy: ~95%

## Usage
Run `DeepfakeCNN.ipynb` in Google Colab. Upload `kaggle.json` when prompted.
For image prediction, upload any face image when prompted.
For video prediction, the notebook constructs a test video from dataset images and runs frame-by-frame inference with majority voting.

## Dependencies
tensorflow
opencv-python
numpy
matplotlib
scikit-learn

## Note
Model is trained on StyleGAN-generated faces. Performance may vary on faces generated by other GAN architectures.
